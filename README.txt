🛠️ Panel de Administración
Este proyecto consiste en un panel de administración moderno y responsivo, que incluye navegación a través de diferentes secciones clave de gestión.

🚀 Tecnologías y Librerías Utilizadas
Framework: React + Vite

Estilos: Tailwind CSS

Ruteo: React Router DOM

Íconos: Lucide React

🧭 Funcionalidades Principales
Navegación entre las siguientes secciones:

🧑‍💼 Clientes

🏢 Proveedores

👥 Usuarios

🔓 Cerrar sesión (Log Out)

Página de inicio (Home) con acceso general.

Diseño responsivo adaptado a diferentes dispositivos.

Modo oscuro (🌙 Dark Mode) activable por el usuario.

Barra de navegación superior (Navbar) responsive e integrada con todas las rutas.
